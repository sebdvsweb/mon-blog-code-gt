Découverte des langages HTML & CSS

  • Seb
  • Date : 8 juin 2023

HTML et CSS à travers CodePen et Visual Studio Code

Codepen.io est un éditeur de code en ligne nous permettant de tester rapidement tous les fragments de code qui vont nous faire découvrir le balisage HTML. Avec une interface simple, le site vous permet d’écrire du code HTML, CSS, et Javascript et de simuler la page que les codes sont censés créer.

Le premier langage étudié est le HTML (HyperText Markup Language). C’est un langage qui permet de structurer des pages web. On parle de langage de balisage et non de langage de programmation, car le but du HTML est d’encadrer les différents éléments présents dans une page (images, titres, paragraphes ...) par des balises.

Exemple de code HTML

Le code ci-dessous crée une balise <p> qui définit un paragraphe auquel on applique une classe CSS. 

<p class="ma_class">"Ceci est un texte"</p>

Le langage CSS (pour Cascading Style Sheets - feuille de style en cascade en VF) est un langage de feuille de styles. Il permet d'apporter du design au contenu HTML intégré. CSS est capable de changer tous les aspects du design d'une page web comme les couleurs, le positionnement ou les polices de caractère et il permet également de créer des animations, de petits effets simples de survol ou d'interaction et bien beaucoup d'autres choses.

Exemple de code CSS

Le code CSS suivant ajoute une couleur de fond et de la marge extérieure à la classe ma_class. En CSS, une classe est un sélecteur qui permet de cibler des éléments d'un document HTML en fonction du contenu de l'attribut class de chaque élément (comme ci-dessus, la balise </p> a pour class ma_class). 

.ma_class { 
background-color: #a6e22e;
margin: 10px;
}

Visual Studio Code est un éditeur de code distribué par Microsoft. C'est l'éditeur de code le plus utilisé au monde. Il permet de créer des fichiers de code en local sur son PC et de structurer l'arborescence de ses dossiers web. Il permet aussi grâce à une extension de simuler un serveur web et d'afficher le résultat de notre code sur un navigateur.

Les langages HTML et CSS nécessitent de respecter une syntaxe bien précise sous peine de connaître des problèmes d'affichage.

Les "variantes" de HTML

HTML, Pug et Haml sont tous des langages de balisage qui permettent de structurer et de donner du sens aux éléments d'une page web. Cependant, ils ont des différences dans leur syntaxe et leur approche.

  • HTML est le langage de balisage standard pour représenter les pages web. Il utilise des balises écrites au format < > pour décrire la structure et le contenu de la page web à afficher
  • Pug, anciennement appelé Jade, est un moteur de templates implémenté en JavaScript qui permet de générer dynamiquement du HTML. Il utilise une syntaxe simplifiée et plus concise que HTML, en utilisant des indentations et des espaces plutôt que des balises
  • Haml est un autre langage de balisage qui utilise une syntaxe encore plus simplifiée que Pug. Il utilise des indentations et des symboles pour représenter les éléments de la page web, plutôt que des balises

En résumé, HTML, Pug et Haml sont tous des langages de balisage qui permettent de structurer et de donner du sens aux éléments d'une page web. HTML est le langage standard, tandis que Pug et Haml utilisent des syntaxes simplifiées pour faciliter la création de pages web.
